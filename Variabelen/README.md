# CSharpPortfolio-WoutPeeters

De aard van variabelen zoals int, double, string, ... 

Het declareren van de variabelen:

```csharp
int getal1;
double getal2;
string tekst;
string[] Array;
```
Een waarde geven aan de variabelen:
```csharp
getal1 = 2;
getal2 = 1.25;
tekst = "Dit is tekst";
```
Of in één keer:
```csharp
int getal = 3;
```
